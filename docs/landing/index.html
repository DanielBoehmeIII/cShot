cShot

Custom one-shot kits from songs, samples, genres, and vibes

Try cShot

How it works

1

Describe

Describe the vibe: “dark rnb one shots” or upload a song/sample

2

Generate

cShot creates 20-200 one-shot sounds tailored to your reference in seconds

3

Curate

Listen, rate, favorite — then export your kit to your DAW

Three ways to create

From a Vibe

cshot make "dark rnb one shot kit"

Describe the sound you want. cShot infers genre, mood, and categories.

From a Song

cshot from-song track.wav

Upload any song. cShot analyzes tempo, key, and texture to create a matching kit.

From a Sample

cshot from-sample clap.wav

Build complementary sounds around one sample for a coherent mini kit.

Install

git clone https://github.com/anomalyco/cShot.git
cd cShot
pip install -r requirements.txt
./cshot make "dark rnb one shot kit" --count 50

Or install to PATH:

pip install -e .
cshot make "alien trap drum kit" --count 40

Demo kits

# Generate a quick demo in seconds
./cshot demo "dark rnb one shots" --count 20

# 5 flagship kits
./cshot build-demo-kits --count 30
