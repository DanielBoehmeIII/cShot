# Week 27 — Video Demo Script

## Title: cShot — Create Custom One-Shot Kits in Seconds

**Duration:** 60-90 seconds

### Script

[0:00] **Intro**
> "Meet cShot — the fastest way to create custom one-shot kits from any song, sample, or vibe."

[0:10] **From a Song**
> Screen recording: drag a song onto terminal
> `cshot from-song beat.wav --count 40`
> "Upload any track. cShot analyzes its tempo, key, and texture — then generates a matching kit."

[0:25] **Browse Results**
> `cshot listen outputs/kits/from_beat/`
> Keyboard: f = favorite, t = trash, space = play
> "Preview every sound with keyboard shortcuts. Favorite what you love, trash what you don't."

[0:40] **From a Vibe**
> `cshot make "dark rnb one shot kit" --count 50`
> "Or just describe what you want. cShot infers the genre, mood, and category balance."

[0:50] **The Result**
> Show organized kit folder: kicks/, snares/, hats/, etc.
> "A polished, categorized, DAW-ready kit in under 10 seconds."

[1:00] **Export**
> `cshot export-daw outputs/kit --daw ableton`
> "Export with proper folder structure and naming for Ableton or FL Studio."

[1:15] **CTA**
> "Try cShot today — open source, local, no cloud needed."
> Link to GitHub

### Production Notes
- Use a real song the viewer would recognize
- Show the terminal with a clean dark theme
- Keep cuts tight — show each command executing in real-time
- End screen: github.com/anomalyco/cShot
