# Prompt 86 — Non-Destructive Sound Design

Every sound in cShot preserves its full history: source, generation parameters, edits, branches, versions, and exports. Nothing is ever lost.

---

## 1. Edit Graph

### Graph Structure

```
Each sound is a DAG (directed acyclic graph) of operations.

                            [Source]
                               │
                    ┌──────────┴──────────┐
                    │                     │
              [Generation]           [Import]
              (prompt, seed,        (file path,
               model, params)        format, origin)
                    │                     │
                    └──────────┬──────────┘
                               │
                         [Original Audio]
                               │
                    ┌──────────┴──────────┐
                    │                     │
              [Edit: EQ]           [Branch: "808 version"]
                    │                     │
                    │               [Edit: Pitch Down -12st]
                    │                     │
                    │               [Edit: Add Sub Osc]
                    │                     │
                    └──────────┬──────────┘
                               │
                        [Edit: Compressor]
                               │
                    ┌──────────┴──────────┐
                    │                     │
              [Export V1]          [Edit: Saturator]
                                         │
                                   [Export V2]
                                         │
                                   [Branch: "live version"]
                                         │
                                   [Edit: Add Reverb]
                                         │
                                   [Export V3]
```

### Graph Data Model

```rust
/// The edit graph stores every operation applied to a sound.
/// It's stored as a JSON document alongside the sound file.

#[derive(Serialize, Deserialize)]
struct SoundGraph {
    /// Unique sound identifier
    sound_id: String,

    /// Root node (source of the sound)
    root: Node,

    /// All nodes in the graph
    nodes: Vec<Node>,

    /// Edges: parent-child relationships
    edges: Vec<Edge>,
}

#[derive(Serialize, Deserialize)]
#[serde(tag = "type")]
enum Node {
    /// Sound was generated by AI
    Generation {
        id: String,
        prompt: String,
        seed: u64,
        model_id: String,
        model_version: String,
        generation_params: HashMap<String, f32>,
        inference_time_ms: u64,
    },

    /// Sound was imported from a file
    Import {
        id: String,
        original_path: Option<String>,
        original_filename: String,
        original_format: String,
        import_timestamp: String,
        checksum: String,
    },

    /// Sound was recorded live
    Recording {
        id: String,
        input_device: String,
        sample_rate: u32,
        bit_depth: u16,
        duration_ms: u64,
        recording_timestamp: String,
    },

    /// DSP Edit
    DspEdit {
        id: String,
        processor: String,       // e.g., "equalizer", "transient_shaper"
        parameters: HashMap<String, f32>,
        intent: Option<EditIntent>,
        bypassed: bool,
    },

    /// Model-based edit
    ModelEdit {
        id: String,
        edit_prompt: String,
        model_id: String,
        strength: f32,
        guidance_scale: f32,
    },

    /// Branch point (logical grouping)
    Branch {
        id: String,
        name: String,
        description: String,
        created_at: String,
    },

    /// Export (terminal node)
    Export {
        id: String,
        format: String,          // "wav", "flac", "mp3", "ogg"
        sample_rate: u32,
        bit_depth: u16,
        channels: u8,
        normalize: bool,
        filename: String,
        exported_at: String,
    },

    /// Tag/Metadata change
    Metadata {
        id: String,
        changes: HashMap<String, String>,
    },
}

#[derive(Serialize, Deserialize)]
struct Edge {
    from: String,
    to: String,
    label: Option<String>,
}
```

### Graph Operations

```rust
impl SoundGraph {
    /// Add a new edit node connected to the current tip
    fn add_edit(&mut self, parent_id: &str, edit: DspEdit) -> String {
        let node_id = generate_uuid();
        self.nodes.push(Node::DspEdit { id: node_id.clone(), ..edit });
        self.edges.push(Edge {
            from: parent_id.to_string(),
            to: node_id.clone(),
            label: None,
        });
        node_id
    }

    /// Create a branch from any node
    fn create_branch(&mut self, from_node_id: &str, name: &str) -> String {
        // Add branch node
        let branch_id = generate_uuid();
        self.nodes.push(Node::Branch {
            id: branch_id.clone(),
            name: name.to_string(),
            description: String::new(),
            created_at: Utc::now().to_rfc3339(),
        });

        // Connect branch point
        self.edges.push(Edge {
            from: from_node_id.to_string(),
            to: branch_id.clone(),
            label: Some("branch".to_string()),
        });

        branch_id
    }

    /// Collect all ancestors of a node (for reconstruction)
    fn ancestors(&self, node_id: &str) -> Vec<&Node> {
        let mut path = Vec::new();
        let mut current = node_id;

        loop {
            // Find the node
            if let Some(node) = self.nodes.iter().find(|n| n.id() == current) {
                path.push(node);

                // Find parent
                if let Some(edge) = self.edges.iter().find(|e| e.to == current) {
                    current = &edge.from;
                } else {
                    break; // Reached root
                }
            } else {
                break;
            }
        }

        path.reverse(); // Root first
        path
    }

    /// Get all leaf nodes (exports that can be flattened)
    fn exports(&self) -> Vec<&Node> {
        self.nodes.iter().filter(|n| matches!(n, Node::Export { .. })).collect()
    }
}
```

## 2. Parameter History

### Per-Parameter History

```rust
/// Track every parameter change over time.
/// Enables: "what was the EQ before I changed it?" and "undo that change".

#[derive(Serialize, Deserialize)]
struct ParameterHistory {
    /// Which sound
    sound_id: String,

    /// Which processor
    processor_id: String,

    /// Which parameter (e.g., "gain", "frequency")
    parameter_name: String,

    /// All changes to this parameter, newest first
    changes: Vec<ParameterChange>,
}

#[derive(Serialize, Deserialize)]
struct ParameterChange {
    timestamp: String,
    old_value: f32,
    new_value: f32,
    change_type: ChangeType,  // Manual, IntentDriven, Preset, Undo, Redo
    intent_id: Option<String>, // Link to the edit intent that caused this
}

impl ParameterHistory {
    /// Apply a parameter change and record it
    fn apply_change(&mut self, new_value: f32, change_type: ChangeType, intent_id: Option<String>) {
        let old_value = self.changes.first()
            .map(|c| c.new_value)
            .unwrap_or(0.0);

        self.changes.insert(0, ParameterChange {
            timestamp: Utc::now().to_rfc3339(),
            old_value,
            new_value,
            change_type,
            intent_id,
        });

        // Trim history (keep last 100 changes per parameter)
        if self.changes.len() > 100 {
            self.changes.truncate(100);
        }
    }

    /// Get value at a specific edit position
    fn value_at(&self, edit_index: usize) -> f32 {
        self.changes.iter()
            .rev()
            .nth(edit_index)
            .map(|c| c.new_value)
            .unwrap_or(0.0)
    }

    /// Get the current value
    fn current_value(&self) -> f32 {
        self.changes.first().map(|c| c.new_value).unwrap_or(0.0)
    }
}
```

## 3. Reversible Processing

### Reversibility by Operation

```
Operation                    Reversible?   Method
───────────────────────────  ────────────  ─────────────────────────
Gain change                  ✓             Store original gain
EQ                           ✓             Store inverse EQ curve
Fade in/out                  ✓             Store original envelope
Trim                         ✓             Store removed samples
Resample                     ✓             Store original sample rate
Time stretch                 ~             Store stretch ratio (imperfect)
Pitch shift                  ~             Store shift amount (imperfect)
Normalize                    ~             Store original peak (rounding)
Compression                  ~             Store compression params (not exact inverse)
Saturation                   ✗             Non-linear, information lost
Clipping                     ✗             Information permanently lost
Noise reduction              ✗             Removed material not stored
Reverb                       ~             Store dry/wet, but convolution is lossy

KEY INSIGHT:
  All edits store the ORIGINAL + PARAMETERS.
  Reversibility = ability to reconstruct original from stored data.
  Not all edits are perfectly reversible — we store snapshots instead.
```

### Audio Reconstruction

```rust
impl SoundGraph {
    /// Reconstruct audio at any node by replaying ancestors.
    fn reconstruct(&self, target_node_id: &str, audio_store: &ContentStore) -> Result<AudioBuffer> {
        let ancestors = self.ancestors(target_node_id);

        // Start with root audio
        let mut audio = match ancestors.first() {
            Some(Node::Generation { .. }) => audio_store.load_generation(ancestors.first().unwrap()),
            Some(Node::Import { .. }) => audio_store.load_original(ancestors.first().unwrap()),
            Some(Node::Recording { .. }) => audio_store.load_recording(ancestors.first().unwrap()),
            _ => return Err("No source node found"),
        }?;

        // Replay all operations in order
        for node in ancestors.iter().skip(1) {
            match node {
                Node::DspEdit { processor, parameters, .. } => {
                    audio = self.apply_dsp(audio, processor, parameters)?;
                }
                Node::ModelEdit { edit_prompt, .. } => {
                    audio = self.apply_model_edit(audio, edit_prompt)?;
                }
                Node::Export { .. } | Node::Branch { .. } | Node::Metadata { .. } => {
                    // No audio change for these nodes
                }
                _ => {}
            }
        }

        Ok(audio)
    }

    /// Fast reconstruction from nearest snapshot.
    fn reconstruct_fast(&self, target_node_id: &str,
                         audio_store: &ContentStore,
                         snapshots: &[SoundSnapshot]) -> Result<AudioBuffer> {
        // Find nearest snapshot before target
        let target_idx = self.node_index(target_node_id);
        let snapshot = snapshots
            .iter()
            .filter(|s| s.edit_index <= target_idx)
            .max_by_key(|s| s.edit_index);

        match snapshot {
            Some(snap) => {
                // Start from snapshot
                let audio = self.reconstruct_at_snapshot(snap, audio_store)?;

                // Apply only edits after snapshot
                let remaining = self.nodes.iter()
                    .skip(snap.edit_index + 1)
                    .take_while(|n| self.ancestors(target_node_id).contains(n));

                for node in remaining {
                    if let Node::DspEdit { processor, parameters, .. } = node {
                        audio = self.apply_dsp(audio, processor, parameters)?;
                    }
                }
                Ok(audio)
            }
            None => self.reconstruct(target_node_id, audio_store),
        }
    }
}
```

### Bypass / A/B Toggle

```rust
impl SoundGraph {
    /// Toggle a specific edit on/off without modifying history.
    fn toggle_edit(&mut self, edit_id: &str) -> Result<AudioBuffer> {
        // Find the edit node
        if let Some(Node::DspEdit { ref mut bypassed, .. }) = self.nodes.iter_mut().find(|n| n.id() == edit_id) {
            *bypassed = !*bypassed;
        }

        // Reconstruct current state with bypass applied
        let current_tip = self.current_tip();
        self.reconstruct_with_bypasses(&current_tip)
    }

    /// Reconstruct while respecting bypass flags.
    fn reconstruct_with_bypasses(&self, target_node_id: &str) -> Result<AudioBuffer> {
        let ancestors = self.ancestors(target_node_id);
        let mut audio = self.load_root_audio()?;

        for node in ancestors.iter().skip(1) {
            match node {
                Node::DspEdit { processor, parameters, bypassed, .. } if !bypassed => {
                    audio = self.apply_dsp(audio, processor, parameters)?;
                }
                Node::ModelEdit { edit_prompt, .. } => {
                    audio = self.apply_model_edit(audio, edit_prompt)?;
                }
                _ => {} // Skip bypassed edits and non-audio nodes
            }
        }

        Ok(audio)
    }
}
```

## 4. Rendered Cache

### Cache Strategy

```
Cache hierarchy for non-destructive editing:

  ┌──────────────────────────────────────────────────┐
  │              L1: In-Memory Cache                   │
  │  • Current state (fully rendered)                 │
  │  • Last 3 branch tips                             │
  │  • Recently played versions                       │
  │  Size: 10-50 MB                                   │
  │  Invalidated: on edit                             │
  └──────────────────────┬───────────────────────────┘
                         │
  ┌──────────────────────▼───────────────────────────┐
  │              L2: Disk Cache (SSD)                 │
  │  • Waveform peaks for each edit state            │
  │  • Spectrogram tiles for visible regions          │
  │  • Snapshot states (every 10 edits)              │
  │  • Before/after diff data                        │
  │  Size: 100-500 MB                                │
  │  Invalidated: on edit, but snapshots persist     │
  └──────────────────────┬───────────────────────────┘
                         │
  ┌──────────────────────▼───────────────────────────┐
  │              L3: Permanent Store                   │
  │  • Export renders (user-committed)                │
  │  • Full snapshots at branch points                │
  │  • Never evicted                                  │
  └──────────────────────────────────────────────────┘
```

### Cache Manager

```rust
struct RenderCache {
    /// In-memory LRU cache
    memory: LruCache<String, AudioBuffer>,

    /// Disk cache directory
    disk_cache_path: PathBuf,

    /// Maximum disk cache size
    max_disk_bytes: u64,

    /// Current disk usage
    current_disk_bytes: AtomicU64,
}

impl RenderCache {
    /// Get rendered audio for a node. Compute if not cached.
    fn get_or_render(&self, node_id: &str, graph: &SoundGraph,
                     audio_store: &ContentStore) -> Result<AudioBuffer> {
        // Check memory cache first
        if let Some(audio) = self.memory.get(node_id) {
            return Ok(audio.clone());
        }

        // Check disk cache
        let cache_path = self.disk_cache_path.join(format!("{}.wav", node_id));
        if cache_path.exists() {
            let audio = AudioBuffer::from_wav(&cache_path)?;
            self.memory.put(node_id.to_string(), audio.clone());
            return Ok(audio);
        }

        // Compute from graph
        let audio = graph.reconstruct_fast(node_id, audio_store, &self.snapshots)?;

        // Cache to disk (async)
        let path = cache_path.clone();
        let buf = audio.clone();
        tokio::spawn(async move {
            buf.to_wav(&path).ok();
        });

        // Cache to memory
        self.memory.put(node_id.to_string(), audio.clone());

        Ok(audio)
    }

    /// Invalidate cache for a subtree (when edits change)
    fn invalidate_subtree(&mut self, node_id: &str, graph: &SoundGraph) {
        // Remove from memory cache
        self.memory.pop(node_id);

        // Mark for disk cache invalidation
        let descendants = graph.descendants(node_id);
        for desc_id in descendants {
            self.memory.pop(&desc_id);
            // Don't delete disk cache immediately — lazy invalidation
            // Disk cache will be overwritten on next render
        }
    }

    /// Pre-cache adjacent branch tips (fast branch switching)
    fn pre_cache_branches(&mut self, graph: &SoundGraph, current_tip: &str) {
        let sibling_branches = graph.sibling_branches(current_tip);
        for branch_tip in sibling_branches {
            let cache_path = self.disk_cache_path.join(format!("{}_pre.wav", branch_tip));
            if !cache_path.exists() {
                let audio = graph.reconstruct(&branch_tip, &self.audio_store);
                if let Ok(audio) = audio {
                    let path = cache_path.clone();
                    tokio::spawn(async move { audio.to_wav(&path).ok(); });
                }
            }
        }
    }
}
```

## 6. Rollback UX

### Rollback Operations

```
Rollback Types:
  1. Undo last edit (Ctrl+Z)
     - Revert to state before most recent edit
     - Edit is preserved in history, just not applied

  2. Rollback to specific version
     - Select any node in the graph → current state becomes that node
     - All edits after that point are bypassed (not deleted)

  3. Revert to original
     - Jump to root node (generation or import)
     - All edits bypassed

  4. Delete branch
     - Remove entire branch from graph
     - WARNING: this is destructive (unlike bypass)
     - Confirmation required: "Delete branch '808 version'? This cannot be undone."

  5. Merge branch
     - Take two branches and combine
     - Conflict resolution: choose which edit wins for conflicting parameters
```

### Rollback Implementation

```rust
impl SoundGraph {
    /// Soft rollback: set current tip to a historical node
    fn rollback(&mut self, target_node_id: &str) {
        self.current_tip = target_node_id.to_string();
        // All edits between target and old current are bypassed, NOT deleted
        // User can "roll forward" to get back
    }

    /// Hard rollback: delete all edits after target
    fn rollback_hard(&mut self, target_node_id: &str) -> Vec<String> {
        let deleted: Vec<String> = self.descendants(target_node_id)
            .iter()
            .map(|n| n.id().to_string())
            .collect();

        // Remove edges
        self.edges.retain(|e| {
            !deleted.contains(&e.from) && !deleted.contains(&e.to)
        });

        // Remove nodes
        self.nodes.retain(|n| !deleted.contains(&n.id()));

        // Set current tip
        self.current_tip = target_node_id.to_string();

        deleted
    }

    /// Merge two branches (simple strategy: last-writer-wins per param)
    fn merge(&mut self, branch_a_tip: &str, branch_b_tip: &str) -> Result<String> {
        let params_a = self.get_parameters(branch_a_tip);
        let params_b = self.get_parameters(branch_b_tip);

        // Merge node: combines both branches
        let merge_node = DspEdit {
            id: generate_uuid(),
            processor: "merge".to_string(),
            parameters: self.merge_params(params_a, params_b, MergeStrategy::LastWriterWins),
            intent: None,
            bypassed: false,
        };

        let merge_id = self.add_edit(branch_b_tip, merge_node);

        // Connect branch A to merge
        self.edges.push(Edge {
            from: branch_a_tip.to_string(),
            to: merge_id.clone(),
            label: Some("merged_from".to_string()),
        });

        Ok(merge_id)
    }

    fn merge_params(&self, a: HashMap<String, f32>, b: HashMap<String, f32>,
                    strategy: MergeStrategy) -> HashMap<String, f32> {
        match strategy {
            MergeStrategy::LastWriterWins => b,  // B's values win
            MergeStrategy::FirstWriterWins => a, // A's values win
            MergeStrategy::Average => {
                let mut merged = a.clone();
                for (key, val_b) in b {
                    let val_a = merged.get(&key).unwrap_or(&0.0);
                    merged.insert(key, (val_a + val_b) / 2.0);
                }
                merged
            }
            MergeStrategy::Union => {
                let mut merged = a.clone();
                merged.extend(b);
                merged
            }
        }
    }
}
```

### Flatten Process

```

into a single audio file with metadata.

  [Generation] → [EQ] → [Compressor] → [Transient Shaper] → [Export]
     prompt      60Hz+3dB   4:1, -18dB     attack+3dB        v1.wav

AFTER (flattened export):
  v1.wav  (2.1 MB, 44.1kHz/24bit/mono)
  v1.metadata.json
  {
    "provenance": {
      "generation": {
        "prompt": "punchy trap kick, 40Hz sub",
        "seed": 42,
        "model": "cshot-v1"
      },
      "edit_chain": [
        {"processor": "eq", "params": {"low_shelf_60hz_gain": 3.0}},
        {"processor": "compressor", "params": {"ratio": 4.0, "threshold": -18.0}},
        {"processor": "transient_shaper", "params": {"attack_gain": 3.0}}
      ],
      "export": {
        "format": "wav",
        "sample_rate": 44100,
        "bit_depth": 24,
        "timestamp": "2026-05-15T12:38:00Z"
      }
    },
    "cshot_id": "sound_abc123",
```

### Export Formats

```rust
struct ExportConfig {
    format: ExportFormat,
    sample_rate: u32,
    bit_depth: u16,
    channels: u8,
    normalize: bool,
    normalize_target: f32,       // LUFS target if normalize
    trim_silence: bool,
    fade_out: Option<FadeConfig>,
    metadata_embed: bool,         // Embed cShot metadata in file
    include_provenance: bool,     // Include JSON sidecar
}

enum ExportFormat {
    Wav,       // Uncompressed, universal
    Flac,      // Compressed, lossless, open
    Mp3,       // Compressed, lossy, universal
    Ogg,       // Compressed, lossy, open
    Aiff,      // Uncompressed, Mac-compatible
}

impl ExportEngine {
    fn flatten_and_export(&self, graph: &SoundGraph, tip_id: &str,
                          config: ExportConfig) -> Result<ExportResult> {
        // 1. Reconstruct audio at tip
        let audio = graph.reconstruct(tip_id, &self.audio_store)?;

        // 2. Apply export processing
        let processed = self.apply_export_processing(audio, &config)?;

        // 3. Encode to target format
        let encoded = self.encode(processed, &config)?;

        // 4. Write file
        let export_path = self.generate_export_path(graph.sound_id, &config);
        std::fs::write(&export_path, &encoded)?;

        // 5. Write metadata sidecar (if enabled)
        if config.include_provenance {
            let metadata = self.generate_metadata(graph, tip_id, &config);
            let meta_path = export_path.with_extension("metadata.json");
            std::fs::write(&meta_path, serde_json::to_string_pretty(&metadata)?)?;
        }

        // 6. Record export in graph
        let export_node = Node::Export {
            id: generate_uuid(),
            format: config.format.to_string(),
            sample_rate: config.sample_rate,
            bit_depth: config.bit_depth,
            channels: config.channels,
            normalize: config.normalize,
            filename: export_path.file_name().unwrap().to_string_lossy().to_string(),
            exported_at: Utc::now().to_rfc3339(),
        };
        graph.nodes.push(export_node);

        Ok(ExportResult {
            path: export_path,
            format: config.format,
            size_bytes: encoded.len() as u64,
            duration_ms: processed.duration_ms(),
        })
    }
}
```

## Summary

1. **Edit graph**: DAG of all operations (generation, DSP edits, model edits, exports) — preserves full provenance
2. **Parameter history**: Per-processor, per-parameter change log with snapshots every 10 edits for fast reconstruction
3. **Reversible processing**: Non-destructive edits store original + parameters; non-reversible ops use full snapshots; any edit can be bypassed
4. **Rendered cache**: Three-level cache (memory → disk → permanent) with LRU eviction and predictive pre-caching
5. **Version browser**: Tree visualization with side-by-side comparison, rollback, merge, branch management
6. **Export flattening**: Converts graph to flat file with embedded provenance metadata — never lose track of how a sound was made
